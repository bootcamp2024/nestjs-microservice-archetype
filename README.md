# Nestjs Microservice Archetype
## Descripción

Arquetipo para aplicar microservicios usando [Nest](https://github.com/nestjs/nest) framework.

## Instalación

```bash
$ pnpm install
```
## Subir infraestructura

```bash
$ docker-compose up
```

## Correr la aplicación

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Pruebas

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
